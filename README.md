# Women say pay, men say culture?
# Topic modeling of Glassdoor reviews
This project is developed in the course Data Science I: Foundations. Original dataset can be found at Kaggle (https://www.kaggle.com/datasets/davidgauthier/glassdoor-job-reviews-2/data).
Analysis is based on the top 15 companies in healthcare (mainly hospitals) and infrastructure. Complete list can be found at https://brightonhps.com/largest-health-systems-in-the-us/ and https://finance.yahoo.com/news/top-15-infrastructure-companies-us-141209351.html
